
### 年齢に基づく自動選択

```javascript
/**
 * 年齢から適切なモデルを自動選択
 */
function selectModel(age, bodyType = 'standard', pointCount = 23) {
  if (age >= 3 && age <= 5) {
    return `yokoi_03_05_${bodyType}`;
  } else if (age >= 6 && age <= 8) {
    return `yokoi_06_08_${bodyType}`;
  } else if (age >= 18 && age < 65) {
    return pointCount === 25 ? 'ae15' : 'ae14';
  } else if (age >= 65) {
    return pointCount === 25 ? 'okada15' : 'okada14';
  }
  return 'ae14'; // デフォルト
}

// 使用例
const method = selectModel(70); // 'okada14'
const calculator = new BodyCenterOfMass(method, 'male');
```

## 🔧 技術仕様

### 計算アルゴリズム

```
1. 仮想ポイントの計算
   大転子中点 = (右大転子 + 左大転子) / 2
   肋骨下端中点 = (右肋骨下端 + 左肋骨下端) / 2  ※25ポイントのみ

2. 部分重心の計算
   部分重心 = 末梢端 × 質量中心比 + 中枢端 × (1 - 質量中心比)

3. 身体重心の計算
   身体重心 = Σ(各セグメントの部分重心 × そのセグメントの質量比)
```

### パフォーマンス

| 処理 | 時間 | 備考 |
|------|------|------|
| 1フレーム計算 | ~0.5ms | 14セグメント |
| 100フレーム計算 | ~50ms | 直列処理 |
| 1000フレーム計算 | ~500ms | 直列処理 |

**メモリ使用量**: 約1MB (1000フレーム)

## ⚠️ 注意事項

### データギャップ

1. **9-17歳のデータがない**
   - この年齢層は成人モデル(阿江)を使用
   - 精度はやや低下する可能性

2. **9歳以上の子どもの体型別データがない**
   - 標準体型のみと仮定

### 適用限界

1. **BMIの極端な値**
   - 体型分類(痩身/標準/肥満)は一般的な範囲を想定
   - 極端なBMIの場合は精度が低下

2. **病理的状態**
   - 筋萎縮症、肥満症などの病的状態には適用不可
   - 通常の健常者を対象としたデータ

3. **民族差**
   - 日本人を対象としたデータ
   - 他の民族には適用時に注意が必要

## 📞 サポート

### よくある質問

**Q1: 9-17歳の身体重心計算の精度は?**
A: 成人モデルを使用するため、やや精度は低下しますが、実用上は問題ない範囲です。

**Q2: 25ポイントモデルと23ポイントモデルの違いは?**
A: 25ポイントは肋骨下端を含み、胴体を上胴と下胴に分割できます。体幹の詳細な分析に有用です。

**Q3: 男女でどれくらい差がありますか?**
A: 質量比で1-3%程度の差があります。女性は脂肪組織が多く、下肢の質量比がやや高めです。

**Q4: 体型分類はどう判断しますか?**
A: BMIまたはローレル指数を基準とします:
- 痩身: ローレル指数 < 115
- 標準: 115 ≤ ローレル指数 ≤ 145
- 肥満: ローレル指数 > 145
