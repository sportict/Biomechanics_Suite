# MotionViewer 角度計算機能 仕様書

## 1. 概要

MotionViewerアプリケーションに、モーションデータから関節角度（Joint Angle）およびセグメント角度（Segment Angle）を計算・表示する機能を追加する。

### 1.1 目的
- 3次元/2次元モーションデータから生体力学的な角度パラメータを自動計算
- リアルタイムでの角度変化の可視化
- 角度データのCSVエクスポート

### 1.2 対象ユーザー
- スポーツバイオメカニクス研究者
- リハビリテーション専門家
- モーション解析技術者

---

## 2. 機能要件

### 2.1 関節角度（Joint Angle）計算

#### 2.1.1 定義
3点（末梢端、関節中心、中枢端）で構成される2つのベクトル間の角度を計算する。

#### 2.1.2 計算方法
```
vec1 = (末梢端 - 関節中心)
vec2 = (中枢端 - 関節中心)
cosθ = (vec1 · vec2) / (|vec1| × |vec2|)
角度 = arccos(cosθ) × (180/π)
```

### 2.2 セグメント角度（Segment Angle）計算

#### 2.2.1 定義
2点（末梢端、中枢端）で構成されるセグメントの、指定された平面における傾斜角度を計算する。

#### 2.2.2 計算方法
```
角度 = atan2(末梢端[cor1] - 中枢端[cor1], 末梢端[cor2] - 中枢端[cor2]) × (180/π)
```

#### 2.2.3 平面選択（3Dデータの場合）
- **X-Y平面**: 正面から見た角度（矢状面）
- **Y-Z平面**: 側面から見た角度（前額面）
- **X-Z平面**: 上から見た角度（水平面）

## 5. 実装詳細

### 5.1 関節角度計算関数

```javascript
/**
 * 関節角度を計算する
 * @param {Object} p1 - 末梢端ポイント {x, y, z}
 * @param {Object} p2 - 関節中心ポイント {x, y, z}
 * @param {Object} p3 - 中枢端ポイント {x, y, z}
 * @param {number} dimension - 次元数 (2 or 3)
 * @returns {number} 角度（度）
 */
calculateJointAngle(p1, p2, p3, dimension) {
  let vec1, vec2, dotProduct, magnitude1, magnitude2;
  
  if (dimension === 2) {
    vec1 = { x: p1.x - p2.x, y: p1.y - p2.y };
    vec2 = { x: p3.x - p2.x, y: p3.y - p2.y };
    dotProduct = vec1.x * vec2.x + vec1.y * vec2.y;
    magnitude1 = Math.sqrt(vec1.x ** 2 + vec1.y ** 2);
    magnitude2 = Math.sqrt(vec2.x ** 2 + vec2.y ** 2);
  } else {
    vec1 = { x: p1.x - p2.x, y: p1.y - p2.y, z: p1.z - p2.z };
    vec2 = { x: p3.x - p2.x, y: p3.y - p2.y, z: p3.z - p2.z };
    dotProduct = vec1.x * vec2.x + vec1.y * vec2.y + vec1.z * vec2.z;
    magnitude1 = Math.sqrt(vec1.x ** 2 + vec1.y ** 2 + vec1.z ** 2);
    magnitude2 = Math.sqrt(vec2.x ** 2 + vec2.y ** 2 + vec2.z ** 2);
  }
  
  const cosTheta = dotProduct / (magnitude1 * magnitude2);
  // 数値誤差対策: -1 <= cosTheta <= 1 にクランプ
  const clampedCos = Math.max(-1, Math.min(1, cosTheta));
  return Math.acos(clampedCos) * (180 / Math.PI);
}
```

### 5.2 セグメント角度計算関数

```javascript
/**
 * セグメント角度を計算する
 * @param {Object} p1 - 末梢端ポイント {x, y, z}
 * @param {Object} p2 - 中枢端ポイント {x, y, z}
 * @param {string} plane - 平面 ('xy' | 'yz' | 'xz')
 * @returns {number} 角度（度）
 */
calculateSegmentAngle(p1, p2, plane) {
  let coord1, coord2;
  
  switch (plane) {
    case 'xy':
      coord1 = p1.x - p2.x;
      coord2 = p1.y - p2.y;
      break;
    case 'yz':
      coord1 = p1.y - p2.y;
      coord2 = p1.z - p2.z;
      break;
    case 'xz':
      coord1 = p1.x - p2.x;
      coord2 = p1.z - p2.z;
      break;
    default:
      coord1 = p1.x - p2.x;
      coord2 = p1.y - p2.y;
  }
  
  return Math.atan2(coord1, coord2) * (180 / Math.PI);
}
```

### 5.3 仮想ポイント計算関数

```javascript
/**
 * 仮想ポイントを計算する
 * @param {Array} frameData - 1フレーム分のポイントデータ
 * @returns {Object} 仮想ポイントを追加したフレームデータ
 */
calculateVirtualPoints(frameData) {
  const rightHip = frameData[13];  // 右大転子 (インデックス13 = ポイント14)
  const leftHip = frameData[19];   // 左大転子 (インデックス19 = ポイント20)
  
  // 大転子中心（仮想ポイント24）
  const hipCenter = {
    x: (rightHip.x + leftHip.x) / 2,
    y: (rightHip.y + leftHip.y) / 2,
    z: (rightHip.z + leftHip.z) / 2
  };
  
  // フレームデータに追加
  const extendedFrame = [...frameData, hipCenter];
  return extendedFrame;
}
```
