# モーションデータ形式

## ファイル形式
- `.sd`: 標準モーションデータ
- `.rd`: 記録データ
- `.2d`: 2Dモーションデータ
- `.3d`: 3Dモーションデータ

## データ構造
```
フレーム数,ポイント数,フレーム間隔(秒)
x1,y1,z1,x2,y2,z2,...,xN,yN,zN
x1,y1,z1,x2,y2,z2,...,xN,yN,zN
...（フレーム数分繰り返し）
```

## サンプルデータ
```
250,29,0.004000
3.806947,1.367895,0.996992,3.698909,1.370197,1.051615,...
3.781558,1.377274,0.989142,3.686801,1.376068,1.040700,...
...
```

## データ仕様
- 座標系: 右手座標系
- 単位: メートル（推定）
- フレーム間隔: 秒単位の浮動小数点
- 座標値: 浮動小数点（6桁精度）
- 典型的な規模: 15-30ポイント、100-1000フレーム
