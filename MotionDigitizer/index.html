3次元CC法（競技場特徴点）

必要カメラ数: カメラ1、カメラ2
最低制御点数: 3点（推奨: 4点以上）
※ 制御点はキャリブレーションテーブルから取得されます
📋 手順ガイド
  1. モードパネルで「キャリブレーション」を選択し、カメラ1・2の動画を読み込む
  2. 下のテーブルで制御点をデジタイズし、実空間座標(X,Y,Z)を入力する
  3. 下の「カメラ初期位置」におおよそのカメラ位置を設定する
  4. 「CC法キャリブレーション実行」ボタンを押す

2次元DLT法（シングル）

必要カメラ数: カメラ1
最低制御点数: 4点
📋 手順ガイド
  1. 「キャリブレーション」モードで、カメラ1の動画を読み込む
  2. テーブルで制御点をデジタイズし、実空間座標(X,Y)を入力する
  3. 「カメラ定数」ボタンを押してキャリブレーションを実行する

2次元DLT法（ステレオ）

必要カメラ数: カメラ1、カメラ2
最低制御点数: 各カメラ4点
📋 手順ガイド
  1. 「キャリブレーション」モードで、カメラ1・2の動画を読み込む
  2. カメラ1→カメラ2の順に制御点をデジタイズし、実空間座標(X,Y)を入力する
  3. カメラ1・2のフレーム範囲を設定する
  4. 「カメラ定数」ボタンを押してキャリブレーションを実行する

3次元DLT法

必要カメラ数: カメラ1、カメラ2
最低制御点数: 6点（推奨: 8点以上）
📋 手順ガイド
  1. 「キャリブレーション」モードで、カメラ1・2の動画を読み込む
  2. テーブルで制御点をデジタイズし、実空間座標(X,Y,Z)を入力する
  3. 「カメラ定数」ボタンを押してキャリブレーションを実行する

4点実長換算

必要カメラ数: カメラ1
必須制御点: マーカー4点（右奥・右前・左前・左奥）
※ オプション: スケール水平左・右、スケール垂直上・下、原点（水平・垂直のスケール検証や座標原点設定に使用）
📋 手順ガイド
  1. 「キャリブレーション」モードで、カメラ1の動画を読み込む
  2. マーカー4点を必ずデジタイズする
  3. マーカー間距離（実測値 [m]）を入力する
  4. 「結果を計算」ボタンを押す

ChArUcoボード法（シングル）

必要カメラ数: カメラ1
必要サンプル数: 20枚以上推奨
📋 手順ガイド
  1. 「キャリブレーション」モードで動画を読み込む
  2. ChArUcoボード設定（縦横数・チェッカーサイズ等）を確認する
  3. 「自動ChArUcoボード検出」でボードが映るフレームを検出する
  4. 「現在フレームを追加」で20枚以上のサンプルを蓄積する
  5. 「キャリブレーション実行」ボタンを押す

ChArUcoボード法（ステレオ）

必要カメラ数: カメラ1、カメラ2
必要サンプル数: 20枚以上推奨
📋 手順ガイド

  3. 「セッション開始」→「自動ChArUcoボード検出」で両カメラの検出を確認する
  4. 「現在フレームを追加」で20枚以上のサンプルを蓄積する
  5. 「キャリブレーション実行」ボタンを押す
